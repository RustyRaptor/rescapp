Create an hybrid MBR inside a GPT partition - Rescapp - Documentation
  • Step 1

    Rescapp searchs for hard disks on your computer.
  • Step 2

    If such hard disks have been found Rescapp presents you the hard disks. At the same time you are prompted:

    Which hard disk?

    You are supposed to choose the hard disk which has a GPT partition which you want to have an hybrid MBR inside it.
  • Step 3

    Rescapp with all gathered information is going to create an hybrid MBR inside your GPT hard disk.Please wait. Currently (Rescatux 0.40b1) this step is time consuming but no Please wait message has been implemented.
  • Step 4

    You are informed about the hybrid mbr creation process.

    Message is self explanatory. You either get:

    Success! Hybrid MBR was created OK! :)

    or

    Failure! Hybrid MBR was not created. Something went wrong! :(
  • Step 5

    A log file called gpt-create-hybrid-mbr_log.txt can be found at Desktop logs folder.
